6 - Récap

La console ?

La console est un outil pour développeur.

Elle permet de tester, debugger en affichant des messages.

Le code n'est exécuté que côté client (dans son propre navigateur).

On l'ouvre avec F12.

Afficher du texte

On peut afficher du texte dans la console avec console.log('texte !');

Ce texte n'est affiché que dans notre console javascript.

On peut aussi utiliser alert('alerte !'); qui affichera le code sur la page dans une popup.

Variables et types

Une variable permet de conserver une valeur, la nommer, et la réutiliser


        let myVar = 'Hello !';
        console.log(myVar); // Affichera 'Hello !' dans la console.

Nous avons pour l'instant vu deux types de variables : number et string.


        let string1 = 'Chaîne de ';
        let string2 = 'caractères';
        let concatenated = string1 + string2;
        console.log(concatenated); // Affichera 'Chaîne de caractères'.

Pour trouver le type d'une variable, on a l'opérateur typeof


        let myString = 'string';
        let myNumber = 42;

        typeof myString; // 'string'
        typeof myNumber; // 'number'

Interaction utilisateur

On peut récupérer une valeur en posant une question à l'utilisateur avec prompt


        let name = prompt('Quel est votre nom ?');
        console.log(name); // La valeur donnée par l'utilisateur

Conversion de types de variables


        typeof parseInt("42"); // "number"
        String(42); // "string"
